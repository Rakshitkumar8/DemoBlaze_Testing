# DemoBlaze_Testing
# DemoBlaze Testing Project (Manual + Automation)

This repository contains a sample manual and automation testing project for the
[DemoBlaze](https://www.demoblaze.com/) e-commerce application.

It includes:

- Manual Testing Artifacts:
  - Test Plan
  - Test Cases (Login + Cart + Basic Checkout)
  - Bug Report
  - Test Summary Report

- Automation Testing (Python + Selenium WebDriver):
  - Login tests (valid & invalid)
  - Add to cart test
  - Place order test

## Tech Stack

- Manual: Excel/Markdown-style documents
- Automation:
  - Python 3.x
  - Selenium WebDriver
  - Google Chrome + ChromeDriver
